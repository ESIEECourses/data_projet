# Data Dashboard

## Description
Ce projet est un tableau de bord interactif développé avec Dash. Il permet de visualiser des données sous forme d'histogrammes et de cartes géographiques.

## Prérequis
- Python 3.8 ou supérieur
- Les packages listés dans `requirements.txt`
